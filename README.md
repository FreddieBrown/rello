# Rello

Command Line Project Management Board

```
USAGE:
    rello [FLAGS] [OPTIONS]

FLAGS:
        --add_item     Add Item to Column
    -h, --help         Prints help information
    -l, --list         List Current Board
        --move_item    Move Item To Different Column
    -V, --version      Prints version information

OPTIONS:
        --add_column <add_column>          Add Column to Board. Pass Column Name.
        --edit_item <edit_item>            Edit Item Data. Pass Item ID
        --remove_column <remove_column>    Remove Column From Board. Pass Column Name.
        --remove_item <remove_item>        Remove Item From Column. Pass Item ID
```
